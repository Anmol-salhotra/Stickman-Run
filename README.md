# 🕹️ Stickman Run

A fast-paced 2D endless runner game built using pure HTML, CSS, and JavaScript. Dodge obstacles, survive as long as possible, and beat your highest score.

🌐 **Play Online:**  
https://anmol-salhotra.github.io/Stickman-Run/

---

## 📖 About

Stickman Run is a browser-based endless runner game developed from scratch without using any game engines or external libraries.

The project was built entirely on a mobile phone as a personal learning project to improve web development and game development skills.

---

## ✨ Features

- 🏃 Endless runner gameplay
- 🎮 Smooth touch and keyboard controls
- 📱 Mobile-friendly design
- 🔊 Sound effects
- ⚡ Increasing challenge as you play
- 🎨 Clean and simple interface
- 🌐 Play directly in your browser
- 🚫 No installation required

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

No frameworks or external libraries were used.

---

## 🚀 How to Play

1. Open the game in your browser.
2. Press the Start button.
3. Jump over incoming obstacles.
4. Stay alive as long as possible.
5. Try to beat your highest score.
